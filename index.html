Coordinate System Conversion

In order to create the transformation matrix, type the vectors in the textarea in the required format! The required format is csv-like, where the first line is a single number, which tells the program the number of dimensions. Then, you need to type dimension + 1 number of vectors in x1;x2;x3;x4 (or more x-es) format for the source vectors, then for the destination vectors. Examples are available here.
The source code is available at the link: https://github.com/bbalage/CoordinateSystemConverter.
If you want to use the previously calculated matrix, type a vector in proper format -- x1;x2;x3 --, then press the button.
